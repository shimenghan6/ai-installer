# AI 装机精灵

一键安装本地大模型，零命令行，双击即用。

## 使用方法

下载 install.ps1，右键运行。
或：
```
python install.py
```

## 做了什么

1. 自动检测硬件（内存和显卡）
2. 推荐最适合配置的模型
3. 自动下载安装 Ollama
4. 自动拉取模型
5. 打开本地聊天页面

## 模型推荐

| 内存 | 推荐模型 |
|------|---------|
| 4GB+ | 千问 0.5B |
| 8GB+ | 千问 3B |
| 16GB+ | 千问 7B |
| 24GB+ | 千问 14B |
| 32GB+ | 千问 32B |

## 环境要求

- Windows 10/11 64位
- Python 3.10+
